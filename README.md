## 🎛 What is pianoterm?

pianoterm is a tool that lets you use a USB MIDI keyboard to run commands on your computer. Instead of typing commands, you press keys on your MIDI keyboard. This can make tasks easier and faster if you prefer playing music keyboards over typing commands.

Though originally made for Linux, this guide will help you set up pianoterm on a Windows PC.

---

## 💻 System Requirements

Before installing pianoterm, make sure your Windows system meets these requirements:

- Windows 10 or later (64-bit recommended)  
- USB port to connect your MIDI keyboard  
- A standard USB MIDI keyboard  
- At least 1 GB of free storage space  
- Administrator rights on your Windows PC  

If your MIDI keyboard needs special drivers, install them before starting.

## 📋 How to Use pianoterm

pianoterm connects your MIDI keyboard to your system’s command line. Here is how you can try it:

### Connect pianoterm to Your MIDI Keyboard

- Once pianoterm is running, it should detect your connected MIDI keyboard automatically.
- You may see a list of recognized MIDI devices. Select your keyboard if asked.

### Assign Commands to Keys

- Pianoterm uses configuration files where you can assign system commands to individual piano keys.
- For example, you could assign one key to open your browser, another key to play music files, or even run scripts.
- The commands run silently in the background when you press a key.

### Using pianoterm Without Coding

- The program might come with sample configuration files with easy instructions.
- To customize, open the configuration file in any text editor (Notepad is fine).
- Each key will have a place to set the command you want to run.

## 🛠 Tips for Working with pianoterm

- Make sure your MIDI keyboard stays connected while you use pianoterm.
- Keep the configuration file backed up in case you need to restore your custom settings.
- You can create different profiles for different tasks and switch between them by loading different configuration files.
- If pianoterm doesn’t detect your keyboard, try unplugging and plugging it back in or restarting the program.

## ⚙ Troubleshooting Common Issues

### pianoterm Won't Start

- Check that your antivirus isn’t blocking the program.
- Confirm that you have the right version for Windows.
- Run the program as Administrator by right-clicking on the executable and selecting “Run as Administrator.”

### MIDI Keyboard Not Recognized

- Confirm the keyboard connects correctly to your PC.
- Try a different USB port.
- Check if the keyboard requires special drivers from the manufacturer.

### Commands Don’t Run When Pressing Keys

- Double-check your configuration file for errors.
- Make sure you saved changes after editing the config.
- Restart pianoterm to reload the configuration file.
